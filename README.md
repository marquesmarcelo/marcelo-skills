# 🚀 Antigravity Skills Hub

Uma coleção aberta de **Skills para Antigravity**, criada para ajudar desenvolvedores, analistas, arquitetos, equipes DevOps e profissionais de IA a acelerarem tarefas do dia a dia utilizando agentes inteligentes.

Este projeto nasceu com um objetivo simples:

> **Compartilhar conhecimento reutilizável em forma de Skills prontas para uso.**

Aqui você encontrará Skills para diferentes cenários, como:

* 🧠 Engenharia de Prompt
* 💻 Desenvolvimento de Software
* ☸️ Kubernetes & DevOps
* 🔐 Segurança da Informação
* 📊 Data Analytics
* 📝 Geração de Documentação
* 🤖 Agentes de IA
* 🧪 Testes e Qualidade
* 📚 Educação e Conteúdo Técnico
* ⚙️ Automação de Tarefas

---

# ✨ O que são Skills?

Skills são conjuntos de instruções, templates, exemplos e comportamentos reutilizáveis que permitem ampliar as capacidades dos agentes de IA.

Com elas, você pode:

* Padronizar respostas
* Criar especialistas virtuais
* Automatizar tarefas repetitivas
* Melhorar a qualidade das entregas
* Compartilhar conhecimento com equipes
* Construir workflows inteligentes

# 🚀 Como Utilizar no Antigravity [https://antigravity.google/]

Clone o projeto:

```bash
git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
```

Copie a Skill desejada para:

```text
meu-projeto/
 ├── .agent/
 │    └── skills/
 │         └── questonildo/
 │              └── SKILL.md
 └── src/
```

ou para o diretório global do Antigravity na sua máquina:

```text
~/.gemini/antigravity/skills/
```

# 🚀 Como Utilizar no Claude Code [https://claude.ai/new]

Clone o projeto:

```bash
git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
```

Você cria uma pasta:

```bash
.claude/skills/
```

Exemplo:

```text
meu-projeto/
 ├── .claude/
 │    └── skills/
 │         └── questonildo/
 │              └── SKILL.md
 └── src/
```

Depois, no prompt do Claude Code, você pede:

Use a skill "questonildo" localizada em .claude/skills/questonildo/SKILL.md

ou:

Leia o arquivo .claude/skills/questonildo/SKILL.md e siga as instruções dele.

O Claude Code consegue ler os arquivos do workspace automaticamente.
